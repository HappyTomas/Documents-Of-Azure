通过JDBC连接MySQL数据方式实现数据库访问，整个系统使用jsp+servlet+service+dao框架构建。
该项目的功能某块较多，其功能模块划分如下：

客户相关：（一组 樊明辉）
客户信息、客户分配、客户关怀、客户类型、客户状态、客户来源、联系记录、联系人

员工相关（二组 王慧勇）
员工信息、房屋信息、房屋类型、部门信息、公告、

邮件相关
写邮件、发件箱、草稿箱

管理员（三组 娄璐盈）
添加员工、添加部门、添加角色

工作台（五组  殷情）
关怀提醒、联系提醒、有效公告、生日提醒

报表（4组 张左）
客户类型报表、客户状态报表、客户来源报表、客户分配报表

客户信息（4组 张左）
查询、编辑、详情、删除

员工信息（4组 张左）
查询、编辑、详情、删除

公告信息（黄增鑫）
查询、删除

发件箱
查询、删除

房屋信息
查询、编辑、删除

发送飞信

个人资料修改
查询、修改
